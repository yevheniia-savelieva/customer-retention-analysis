# Customer Retention Cohort Analysis

## 📊 Project Overview
This project analyzes customer retention using cohort analysis to understand user behavior over time.

Dataset includes user activity over multiple months.

---

## 🎯 Objective
- Analyze user retention by cohort
- Identify trends in user engagement
- Evaluate how retention changes over time

---

## 🧪 Methodology
- Group users into cohorts based on first activity date
- Calculate retention rate for each cohort over time
- Visualize retention patterns

---

## 📈 Results
- Retention decreases over time, which is expected behavior
- Some cohorts perform better, indicating possible product or marketing impact

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn

---

## 📁 Files
- `retention_analysis.ipynb` — full analysis
- PDF report with results
- `sql_cohort_analysis_logic.pdf` — SQL queries and data preparation
