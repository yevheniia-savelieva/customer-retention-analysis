# Customer Retention Cohort Analysis

## 📊 Project Overview
This project analyzes customer retention using cohort analysis to understand user behavior over time.

The dataset includes user activity across multiple months and allows tracking retention trends by cohort.

---

## 🎯 Objective
- Analyze user retention by cohort  
- Identify trends in user engagement  
- Evaluate how retention changes over time  

---

## 🧪 Methodology
- Group users into cohorts based on their first activity date  
- Calculate retention rate for each cohort across multiple months  
- Segment users by promo_signup_flag (promo vs non-promo users)  
- Visualize retention patterns using cohort tables and heatmaps  

---

## 📈 Results
- Retention decreases over time, which is expected user behavior  
- Promo users show faster drop-off compared to non-promo users  
- Some cohorts perform better, indicating potential marketing or product impact

## 📈 Business Insights
- Users acquired through promotions tend to have lower long-term retention  
- The largest drop in retention occurs between month 1 and month 2  
- Retention stabilizes after month 3  

---

## 💡 Recommendations
- Reduce reliance on discounts for acquiring long-term users  
- Improve onboarding experience to increase early retention  
- Focus on user engagement within the first 7 days

## ✅ Conclusion
Cohort analysis reveals that while promotions increase short-term acquisition, they may negatively impact long-term retention. Strategic improvements in onboarding and engagement can significantly enhance user retention.
